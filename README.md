## X-Auth 3

### Usage

Compile code with `sbt compile`, run it with `sbt xauthApi/run`, and `sbt console` will start a Scala 3 REPL.

### Run Tests

`sbt`
`project xauthUtil`
`test`

or

`sbt xauthUtil/test`
